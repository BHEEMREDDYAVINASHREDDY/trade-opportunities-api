# Trade Opportunities API

A FastAPI-based backend service that analyzes market trends and generates trade opportunity insights for different sectors in India using AI.

---

## Overview

This application provides an API endpoint that accepts a sector name and returns a structured market analysis report. It integrates external data sources with the Gemini API to generate meaningful insights.

---

## Features

- FastAPI-based REST API
- JWT Authentication (guest token)
- Rate limiting to prevent misuse
- AI-powered analysis using Gemini API
- Real-time data fetching (DuckDuckGo API)
- Structured Markdown reports
- Deployed on Render

---

## Tech Stack

- Python 3
- FastAPI
- Uvicorn
- HTTPX
- PyJWT
- Python-dotenv
- Google Gemini API
